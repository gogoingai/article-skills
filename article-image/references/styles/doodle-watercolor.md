# 水彩涂鸦风格参考库（doodle-watercolor-style）
> 属于 article-image 技能的风格库家族，见 [SKILL.md](../../SKILL.md) 总览；通用画图规范见 [core-principles.md](../core-principles.md)、[pitfalls.md](../pitfalls.md)。


> 参考图存放于 `../../assets/styles/doodle-watercolor/`

第五种手绘风格家族，与手绘插画风格（[handdrawn.md](handdrawn.md)）、Excalidraw 风格（[excalidraw.md](excalidraw.md)）、单色马克笔风格（[mono-marker.md](mono-marker.md)）都不同——核心特征是**水彩晕染质感图标 + 装饰性涂鸦点缀（星星/螺旋线/火花）+ 彩色波浪箭头**，视觉上像用水彩笔+马克笔混合手绘的概念解说图，比手绘插画风格更"随性涂鸦"、比单色马克笔风格色彩更丰富。

| 特征 | 描述 |
| --- | --- |
| 背景 | 纯白（不要米黄色、不要纸张纹理——这是和 mono-marker 风格的关键区别） |
| 图标 | 水彩晕染质感，线条松散随性，非精细工笔 |
| 装饰元素 | **标志性特征**：图标周围点缀星星、螺旋线、火花、小圆点等涂鸦元素，营造随手涂画的氛围 |
| 箭头 | 手绘波浪曲线箭头，**全图粗细/画法必须统一**，禁止直线箭头和波浪箭头混用；颜色不要每根都换一种，同一张图里最多一到两种色系即可，不要为了"呼应目标图标"而每根箭头单独配色（踩坑，见下方） |
| 标题 | 粗体手写字体，下方配一条彩色手绘下划线装饰 |
| 标签 | 图标下方或右侧居中黑色手写体文字 |
| 分支标注 | 判断分支的"是/否"文字必须显式标注在对应箭头旁，容易被生成遗漏，需要在 prompt 里强调 |
| 构图 | 横向流程用 16:9 宽屏；竖向流程/分层架构用 9:16 竖版——生成前先确定好方向，别让模型自己选 |

**五种手绘风格家族的选择依据**：
- 用户提到「插画」「可爱」「卡通」「图标」→ [handdrawn.md](handdrawn.md) 手绘插画风格
- 用户提到「excalidraw」「白板」「斜线填充」「工程草图感」→ [excalidraw.md](excalidraw.md) Excalidraw 风格
- 用户提到「马克笔」「单色」「波浪线」→ [mono-marker.md](mono-marker.md) 单色马克笔风格
- 用户提到「水彩」「涂鸦」「星星火花装饰」「暖色手绘」→ 本文件

**② prompt 风格描述**，开头必须加：

```
手绘涂鸦风格：水彩晕染质感的手绘图标，线条松散随性，图标周围点缀星星、螺旋线、火花等装饰性涂鸦元素。背景纯白（不要米黄色、不要纸张纹理）。标题用粗体手写字体，下方配一条彩色手绘下划线装饰。全图所有箭头必须是同一种风格：手绘波浪曲线箭头，粗细一致，颜色统一用一到两种色系，不要每根箭头单独换色。
```

### 踩坑记录

生成判断分支类图时容易出现两个问题（已在 `doodle-watercolor-04` 定稿前反复出现）：

1. **箭头风格不统一**：同一张图里出现波浪箭头和直线箭头混用，或箭头颜色乱跳（每根配一种色）。修复方法：prompt 里必须显式声明"全图箭头必须是同一种风格，粗细一致，颜色统一用一到两种色系"，只写"波浪箭头"不够，还要明确禁止直线箭头混入、禁止每根箭头单独配色。
2. **"是/否"分支标签被遗漏**：修复后箭头统一了，但分支判断的文字标签容易在同一轮修改里被模型省略。需要在 prompt 里同时保留"箭头旁标注文字「是」/「否」"的显式要求，不能只强调箭头风格而漏了标签要求。

### Prompt 模板片段

**横向多阶段流程** — 参考 `doodle-watercolor-01`/`02`：
```
手绘涂鸦风格：水彩晕染质感的手绘图标，线条松散随性，图标周围点缀星星、螺旋线、火花等装饰性涂鸦元素。背景纯白。标题用粗体手写字体，下方配一条彩色手绘下划线装饰。
横向宽屏构图，宽高比约 16:9。标题「[标题]」。从左到右 [N] 个阶段，波浪箭头串联（箭头颜色统一用一到两种色系）：阶段一「[名称]」（[图标]+[图标]）；阶段二「[名称]」（[图标]）；…
每个阶段下方居中黑色手写体标签文字。
```

**竖向分层架构** — 参考 `doodle-watercolor-03`：
```
手绘涂鸦风格：水彩晕染质感的手绘图标，线条松散随性，图标周围点缀星星、螺旋线、火花等装饰性涂鸦元素。背景纯白。标题用粗体手写字体，下方配一条彩色手绘下划线装饰。
**竖版构图，宽高比约 9:16，禁止横版**。从上到下 [N] 层，波浪箭头纵向连接（颜色统一用一到两种色系）：第一层「[层名]」（[图标]，浅色圆角底框）；第二层「[层名]」（[图标]）；…
每层右侧居中黑色手写体层名标签。
```

**竖向判断分支流程** — 参考 `doodle-watercolor-04`：
```
手绘涂鸦风格：水彩晕染质感的手绘图标，线条松散随性，图标周围点缀星星、螺旋线、火花等装饰性涂鸦元素。背景纯白。标题用粗体手写字体，下方配一条彩色手绘下划线装饰。
**竖版构图，宽高比约 9:16，禁止横版**。顶部「[起始节点]」（[图标]）→波浪箭头向下→判断菱形「[条件]？」（[图标]）。菱形左侧波浪箭头标注文字「否」，指向分支「[否分支结果]」（[图标]）。菱形正下方波浪箭头标注文字「是」，指向「[是分支后续]」（[图标]）→波浪箭头向下→「[终点]」（[图标]）。
**全图所有箭头必须是同一种风格：手绘波浪曲线箭头，粗细一致，颜色统一用一到两种色系，不要每根箭头单独配色**。两个分支路径不要互相连接，分支之间保持足够宽的间距。
```
