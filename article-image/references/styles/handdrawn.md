# 手绘插画风格参考库（handdrawn-style）
> 属于 article-image 技能的风格库家族，见 [SKILL.md](../../SKILL.md) 总览；通用画图规范见 [core-principles.md](../core-principles.md)、[pitfalls.md](../pitfalls.md)。


> 参考图存放于 `../../assets/styles/handdrawn/`

这套风格适合技术文章配图、社交媒体传播图、流程说明图。核心特征：

| 特征 | 描述 |
| --- | --- |
| 线条感 | 圆角矩形有轻微手绘线条感，非完全机械规整 |
| 字体 | 略带手写质感，中文圆润、标题加粗 |
| 背景 | 纯白或暖白，整体明亮通透 |
| 图标 | 鲜明活泼，是全图最鲜艳的元素，与浅色底色形成反差 |
| 配色 | 各步骤用不同浅色底色（浅蓝/浅黄/浅橙/浅绿/浅紫），标题或数字圆圈用深色 |
| 标题栏 | 圆角矩形横幅，两侧带「——」装饰横线 |
| 步骤编号 | 填色圆圈（①②③…），左上角或正上方，颜色比卡片底色深一档 |

### 生成规则（--ref + prompt 缺一不可）

**① `--ref` 优先**：传入参考图锚定视觉风格（有机线条、水彩填色感），仅靠 prompt 描述无法稳定复现。推荐参考图：

```
--ref ~/.agents/skills/article-image/assets/styles/handdrawn/handdrawn-01-skill-creation-loop.png
```

（也可换 `handdrawn-02` 至 `handdrawn-07`，选和目标图类型最接近的，见下方索引）

**② prompt 风格描述不可省略**：`--ref` 锁视觉质感，prompt 里的风格词控制模型对内容的理解方向，两者共同作用。prompt 开头必须加：

```
手绘插画风格：圆角矩形有轻微手绘线条感，图标鲜明活泼，字体略带手写质感，整体明亮通透，背景纯白。
```

**完整调用示例：**

```bash
RAND=$(openssl rand -hex 4)
bash ~/.agents/skills/article-image/scripts/gpt-image-2-gen.sh \
  --prompt "手绘插画风格，有机线条感，水彩填色质感，背景纯白。[图内容描述...]" \
  --ref ~/.agents/skills/article-image/assets/styles/handdrawn/handdrawn-01-skill-creation-loop.png \
  --out /tmp/article-img-${RAND}.png \
  --timeout-sec 300
```

### 各图类型对应的 prompt 模板片段

**横向线性流程（含判断菱形）** — 参考 `handdrawn-01`：
```
横向宽屏构图，宽高比约 16:9，禁止竖版，背景纯白。
手绘插画风格：圆角矩形有轻微手绘线条感，图标鲜明活泼，字体略带手写质感，整体明亮通透。
顶部标题横幅（圆角矩形，两侧有「——」装饰横线）：「[标题]」
主流程从左到右：步骤①「[名称]」（[浅色]底圆角卡片，左上角[深色]填色圆圈数字①，[图标]），副标「[说明]」；→；步骤②…
判断菱形「[条件]？」（白底[颜色]边框），是→右侧虚线框「[结果]」止于此，否↓继续。
底部从末步骤引出向左弧形虚线箭头回到步骤①下方，配小字「[注释]」
所有步骤卡片横向均匀排列，间距宽松，图标是全图最鲜艳的元素。
```

**竖向三层架构** — 参考 `handdrawn-04`：
```
手绘插画风格，背景纯白，整体明亮通透。
顶部居中大号标题：「[标题]」，下方细横线装饰。
三个大圆角矩形从上到下串联，层间实线箭头带标注文字：
  层一（浅蓝底）：左侧大图标 + 层名（加粗大字） + 左下副标；右侧竖虚线分隔；右区[浅色]小标签「[功能标签]」+ 说明文字。
  层二（浅黄底）：同结构。
  层三（浅橙/浅红底）：同结构。
右侧有竖向大箭头 + 标注文字说明整体趋势。
底部虚线圆角框 + 灯泡图标 + 总结一句话。
```

**环形流程** — 参考 `handdrawn-02`：
```
手绘插画风格，背景纯白。
中央区域空白，5 个带编号圆圈的圆角卡片沿圆形路径排列，弧形箭头依次串联（顺时针）：
  卡片①「[名称]」（[图标]）→ 卡片② … → 卡片⑤ → 弧形箭头回到卡片①。
底部：大圆角虚线矩形标注「[来源/输入]」，内含横向排列的多个小卡片（带图标），箭头引向对应的环形步骤。
右侧：图例小框（[颜色]小方块 + 说明文字）。
```

**中心辐射思维导图** — 参考 `handdrawn-05`：
```
手绘插画风格，背景纯白，整体留白充裕。
顶部居中标题：「[标题]」，下方细横线装饰。
中央大圆角矩形（白底深色边框）：大号文字「[核心词]」，下方小字「[副标]」。
四角各延伸一条弧线连到圆角卡片（左上/左下/右上/右下）：
  左上「[分支A]」（[颜色]底），卡片右侧带图标；
  左下「[分支B]」（[颜色]底），卡片右侧带图标；
  右上「[分支C]」（白底），卡片内图标横向排列；
  右下「[分支D]」（白底），卡片内图标横向排列。
底部：圆角横幅（带[图标]），「[总结一句话]」。
```
