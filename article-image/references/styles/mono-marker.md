# 单色马克笔风格参考库（mono-marker-style）
> 属于 article-image 技能的风格库家族，见 [SKILL.md](../../SKILL.md) 总览；通用画图规范见 [core-principles.md](../core-principles.md)、[pitfalls.md](../pitfalls.md)。


> 参考图存放于 `../../assets/styles/mono-marker/`

第三种手绘风格家族，与手绘插画风格（[handdrawn.md](handdrawn.md)）、Excalidraw 风格（[excalidraw.md](excalidraw.md)）都不同——核心特征是**全图单一主色调 + 奶油色背景 + 波浪线引出注释**，视觉上更像用马克笔在米色笔记本上手绘的图解。

| 特征 | 描述 |
| --- | --- |
| 背景 | 米白色/奶油色（非纯白），带轻微纸张质感 |
| 配色 | **全图只用一种主色**（如绿色/蓝色）：节点描边、节点填充、连接箭头、分组外框全部同色系，只有极少数"类别标签"节点例外用灰色 |
| 节点 | 圆角胶囊/椭圆形，同色系描边+填充（填充可以是浅色调，不强制饱和实心） |
| 分组框 | 粗描边圆角矩形/胶囊形状，内部无填充（透明），把一组节点框在一起 |
| 线条 | 手绘感圆润流畅，不是锯齿状潦草描边（区别于 Excalidraw 的 rough.js 感） |
| 注释连接 | **标志性特征**：节点向右侧引出一条卷曲波浪线（形似手绘的 `~`），末端带箭头，连接到黑色手写说明文字；左侧/其他方向的说明用直线小箭头即可 |
| 字体 | 手写体，笔画偏粗饱满 |

**三种手绘风格家族的选择依据**：
- 用户提到「插画」「可爱」「卡通」「图标」→ [handdrawn.md](handdrawn.md) 手绘插画风格
- 用户提到「excalidraw」「白板」「斜线填充」「工程草图感」→ [excalidraw.md](excalidraw.md) Excalidraw 风格
- 用户提到「马克笔」「单色」「波浪线」→ 本文件

### 生成规则（--ref + prompt 缺一不可，--ref 更关键）

**① --ref 必须**：这套风格的"波浪线注释"「同色系分组框」等细节仅靠文字描述很难稳定复现，务必传参考图：

```
--ref ~/.agents/skills/article-image/assets/styles/mono-marker/mono-marker-01-gateway-flow.png          # 竖向分组流程
--ref ~/.agents/skills/article-image/assets/styles/mono-marker/mono-marker-02-branch-decision.png       # 含判断菱形的分支合并
--ref ~/.agents/skills/article-image/assets/styles/mono-marker/mono-marker-03-layered-arch.png          # 多层横向分层架构
--ref ~/.agents/skills/article-image/assets/styles/mono-marker/mono-marker-04-sync-async-comparison.png # 左右并排对比
```

**② prompt 风格描述**，开头必须加：

```
手绘风格技术示意图，背景为米白色/奶油色（非纯白），略带纸张质感。整体单色调设计：所有节点用同一种颜色的圆角胶囊/椭圆表示，描边也是同色系，线条手绘感圆润流畅（不是潦草锯齿状）。大分组外框用同一种颜色、更粗的描边圆角矩形框住一组节点，内部无填充。节点之间用同色系手绘箭头连接。右侧补充说明通过一条卷曲的波浪线（末端带箭头）从节点引出，连接到黑色手写说明文字。文字手写体风格，笔画偏粗。
```

**⚠️ 配色可替换**：主色可以换成蓝、橙、紫等任意单一颜色，只要保证"全图同一色系"这条规则不破——一旦引入第二种高饱和度颜色（灰色类别标签除外），就偏离了这套风格的识别特征。

### Prompt 模板片段

**竖向分组流程（含波浪线注释）** — 参考 `mono-marker-01`：
```
手绘风格技术示意图，背景为米白色/奶油色，略带纸张质感。整体单色调设计（用[颜色]），所有节点为同色系圆角胶囊，描边同色系，线条手绘感圆润流畅。大分组用同色系粗描边圆角矩形框住一组节点，内部无填充，标签写在框内左上角（灰色胶囊小字）。节点间用同色系手绘箭头连接。右侧补充说明通过卷曲波浪线（末端带箭头）从节点引出连接到黑色手写说明文字。

画面内容：横向/竖向宽屏构图。顶部「[入口节点]」（[颜色]胶囊节点），箭头向下进入一个[颜色]粗描边分组框（标签「[分组名]」）。框内纵向串联 [N] 个[颜色]节点：「[步骤1]」→「[步骤2]」→…，每个节点右侧波浪线引出说明文字：「[说明1]」「[说明2]」…
分组框下方箭头指向底部横向排列的 [N] 个[颜色]小节点：「[分支1]」「[分支2]」…，右侧波浪线引出说明「[整体说明]」。

所有文字用中文。
```
